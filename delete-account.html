Request Account Data Deletion

Users of LifeAlign App can request deletion of their account and associated data at any time.

How to request deletion

Please send an email to: ckmhelp2025@gmail.com

Include the following details:

  • Email address used to sign in (Google Sign-In)
  • Subject line: Account Deletion Request

What data will be deleted

  • User account information (email, profile identifier)
  • Health progress and activity data
  • App usage data associated with the account

Data retention

All user data will be permanently deleted within 30 days of receiving the request, unless retention is required by law.
